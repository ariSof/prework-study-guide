# Prework Study Guide

## Description

BootCamp Prework Study Guide for Students.

Contains notes on topics covered during the Prework: HTML, CSS, JavaScipt, and Git.  Built for bootcamp students to refer back  and review some fundamentals. It is the go to website for when you need to look back on something learned during the Prework section of the bootcamp. 

## Installation

N/A

## Usage
Each section of the Prework Study Guide includes notes on each topic for HTML, CSS, JavaScript, and Git.
To see the JavaScript in action, will need to open Developer Tools. Using chrome, open DevTools by using short cut Ctrl+Shift+I, once that opens select Console. Refresh page to see javascript update suggestion on which subject to study next.
