Prework Study Guide

HTML

  • The head element contains information about the webpage and will not be displayed on the page.
  • Tags that can go inside head are title, style, abse, link, meta, script, and noscript
  • The meta tag goes inside the head, and is used to specify character set, page description, keywaords, auther of document and viewport settings
  • The title tag goes inside the head element, and is required
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A : stage all files to record changes
  • git commit -m "description of change in quotes"
  • git pull origin main: make sure we have the latest branch
  • git push origin name-of-new-branch-feature/name
